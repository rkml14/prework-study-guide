HTML

  • The head element contains information or metada about the webpage
  • The body element represents the visible content show to the user
  • The html opening tag represents the root or start of our HTML document
  • An html attribute defines addtional information about an element
  • The lang="en" inside the html tag tells the browser the language for the website
  • The style element defines the CSS styles associated with a website
  • The base element defines the base URL for a webpage
  • The meta element defines metadata such as character set, description, keywords, author and viewport
  • The charset element sets the character encoding
  • The http-equiv element sets an HTTP header for content
  • The viewport element sets the default view for the webpage
  • The link element connects an external resource to the HTML document
  • Elements nested inside other elements are called Child Elements, inside of a Parent Element
  • The ul element is for unordered lists
  • The ol element is for ordered lists
  • The li element is for list items
  • The img element contains information about images displayed on the webpage. The src attribute defines location of the image
  • The alt attribute for the img element contains a text string that describes the appearance and functionality of the image.
  • The a element stands for anchor, and creates links to the same webpage or other webpages. The href attribute points to the URL for the link
  • The h element is for headlines, in size order from 1 to 6. There should only be one h1 heading on each page

CSS

  • There are 3 ways to style: inline CSS, internal CSS style sheet, external CSS style sheet
  • Inline CSS effects an element directly in a HTML file. Quick & permanent but each must be edited individually for a style change
  • Internal CSS style sheet is embedded directly in the head element. Its rules apply only to that particular page
  • External CSS style sheet allows developers to keep all CSS rules in a seperate file, making design changes easier
  • A link element is placed in the HTML head element to use/link an external style sheet
  • A period (.) preceding card designates it as a class selector. Any HTML element assigned to the class card will gain these properties.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A border property assigns a border width
  • A box-shadow adds a shadow effect around an element's frame

Git

  • ls: allows us to see a list of files and directories on our computer
  • cd: allows us to change directories
  • pwd: print working directory of the current directory in to the terminal
  • cd .. : to navigate back one directory at a time
  • mkdir: to create a directory
  • touch: to create a new file ex: touch first-day.txt
  • checkout: to move the working branch to a new branch
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: to commit all files in the working branch
  • git commit -m "descriptive-text" : to associate a message with our commit
